# Python Code Generator

**Автор:** [Злотин Ефим Дмитриевич] [501620]  
**IDE:** Visual Studio Code  
**Язык:** Python

## Описание
Плагин для автоматической генерации шаблонов кода на Python.

## Возможности
- Вставка if construction
- UPPERCASE выделенного текста
- Вставка for construction  
- Вставка while construction
- Вставка function
- Вставка class

## Горячие клавиши
- Cmd+Shift+I - вставить if
- Cmd+Shift+F - вставить for
- Cmd+Shift+U - вставить UPPERCASE
