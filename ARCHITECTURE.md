# Архитектура плагина Python Code Generator

## Обзор
Плагин для VS Code, предоставляющий инструменты для автоматической генерации Python кода и работы с текстом.

## Структура проекта


## Компоненты системы

### 1. Модуль активации (`activate`)
- **Назначение**: Инициализация расширения при запуске VS Code
- **Функции**:
  - Регистрация всех команд расширения
  - Подписка на события жизненного цикла
  - Инициализация состояния плагина

### 2. Команды плагина

#### Генераторы кода:
- `insertIf` - Вставка условного оператора
- `insertFor` - Вставка цикла for
- `insertWhile` - Вставка цикла while  
- `insertFunction` - Вставка определения функции
- `insertClass` - Вставка определения класса

#### Инструменты текста:
- `toUpperCase` - Преобразование регистра

## Поток данных

### Сценарий использования:
1. **Активация**: VS Code загружает расширение
2. **Регистрация**: Все команды регистрируются в системе
3. **Ожидание**: Плагин ожидает вызова команд пользователем
4. **Обработка**: При вызове команды:
   - Проверяется контекст (тип файла, выделение)
   - Выполняется соответствующее действие
   - Обновляется интерфейс редактора
5. **Обратная связь**: Пользователь получает результат или сообщение об ошибке

## Обработка ошибок

- Проверка активного редактора
- Валидация типа файла (.py только для генераторов кода)
- Проверка выделенного текста (для текстовых операций)
- Информативные сообщения об ошибках

## Зависимости

- **VS Code API**: Основной API для работы с редактором
- **TypeScript**: Язык разработки
- **Node.js**: Среда выполнения
